# SMS Test Tasks

Тестовое задание для позиции Middle Fullstack C# Developer.

## Структура решения
- **Sms.Test.Core** - Общие модели и интерфейсы.

- **Sms.Test.ConsoleApp** - Задание №1 (Консоль + БД).
- **Sms.Test.WpfApp** - Задание №2 (Редактор переменных среды).
